# Original User Request

## Initial Request — 2026-06-10T16:48:54+08:00

开发一个基于 WPF (.NET 10.0-windows) 的快速启动器（AltRunSharp），支持全局快捷键唤醒、搜索过滤、快速启动程序、带参数脚本执行以及设置托盘和配置界面。

Working directory: c:/D/VSProjects/AltRunSharp
Integrity mode: development

## Requirements

### R1. 全局快捷键与窗口唤醒
- 支持在当前活跃显示器的画面中间弹出启动器输入框。
- 默认快捷键为 Alt+R（可修改）。
- 需支持配置双击或三击 Ctrl 键启动。
- 双击/三击 Ctrl 等按键检测应使用全局键盘钩子（Keyboard Hook）或其他可靠的 Win32 API / 第三方库实现。

### R2. 搜索过滤与程序/脚本运行
- 输入框支持模糊搜索已配置的快捷命令/程序名。
- 支持上下键选择搜索结果，按 Enter 键运行选中的命令/程序，并关闭（隐藏）主输入框。
- 脚本执行逻辑：
  - 允许添加命令、详情并编写 JS 或 C# 脚本，脚本保存在 `data\scripts` 目录下。
  - JS 脚本自动使用 `node` 运行，C# 脚本使用 `dotnet run` 运行（使用一个临时的 .csproj 项目文件或 dotnet-script/csc 编译运行，确保可以在无 VS 完整环境下运行）。
  - 支持为脚本传入 args，支持复杂参数解析：
    - 支持带双引号、转义、多 args、带空格的单个 arg，例如：`/command "1234567" "\"1 2\""`
    - 支持不带双引号的、多个不需要转义的 args，例如：`/command 1234567 12345`
  - 运行方式支持“静默运行”与“显式运行”可选：
    - 无论是静默还是显式，均需重定向标准输出（stdout）和标准错误（stderr）。
    - 静默运行：在后台静默执行，输出日志保存到 `data\logs\yyyyMMdd.log`。
    - 显式运行：提供一个美化的输出展示窗口/界面，展示运行过程中的实时输出信息。

### R3. UI 界面设计
- 采用经典的“顶栏 - 左栏 - 右栏”布局。
- **顶栏**：包含程序名称和隐藏到系统托盘的按钮。
- **左栏**：包含以下三个主要功能切换标签：
  - **快速启动**：用于添加/修改/删除快捷命令（包含程序路径、参数、快捷命令名称、详情描述）。
  - **脚本执行**：用于添加/修改/删除脚本命令（包含命令名称、详情描述、脚本类型 JS/CS、以及对应的脚本代码编辑器/输入区域）。
  - **配置页面**：
    - 为 exe 程序添加右键菜单“添加到 AltRun”（开关）。
    - 开机运行（开关）。
    - 全局绑定按键（可配置自定义快捷键，以及开启双击/三击 Ctrl 启动的开关）。
- **右栏**：根据左栏选中的标签展示对应的编辑、列表或配置详情区域。

### R4. 系统集成与权限管理
- 配置文件及数据统一保存在 `.\data\config.json` 中。
- 开启/关闭右键菜单“添加到 AltRun”和开机自启开关时，如果当前进程没有管理员权限，需提示并向用户请求管理员权限（例如，以管理员身份重新启动自身或启动一个提升权限 of helper 进程来修改注册表）。

## Acceptance Criteria

### 基础运行与热键 (R1 & R3)
- [ ] 启动后，系统托盘区域显示 AltRun 托盘图标，右键托盘图标可退出程序。
- [ ] 按下全局快捷键（默认 Alt+R，或在当前活跃屏幕上按下双击/三击 Ctrl）能在当前鼠标所在的显示器屏幕中央弹出输入框。
- [ ] 在输入框外点击或按 Esc 键可以隐藏输入框。

### 搜索、程序与脚本执行 (R2)
- [ ] 在输入框中输入字符，能即时模糊搜索出所有匹配的快速启动程序 and 脚本命令。
- [ ] 支持通过键盘上下方向键切换选中项，按 Enter 键运行。
- [ ] 执行常规 exe 程序时，支持带参数启动。
- [ ] 执行脚本（以 `/` 开头）时：
  - 能够正确解析命令行参数（包含引号、转义、空格等复杂组合）。
  - JS 脚本能够成功调用本地 `node` 执行，C# 脚本能成功通过 `dotnet` 运行。
  - 静默执行的脚本在 `data\logs\` 目录下生成以当天日期命名的 `.log` 文件，并记录 stdout/stderr 内容.
  - 显式执行的脚本会弹出一个精心美化的终端/输出窗口，实时打印 stdout/stderr 并不阻塞主界面。

### 配置及系统集成 (R4)
- [ ] 配置信息持久化在 `data\config.json` 中，程序关闭并重新打开后配置依然生效。
- [ ] 在配置界面开启“添加到 AltRun”开关时，能够成功将 exe 右键菜单写入注册表；如果权限不足，程序应弹出 UAC 提权请求。
- [ ] 开启“开机自启”开关时，能够将程序路径写入 Windows 注册表启动项或创建启动任务，权限不足时能正确提权。
